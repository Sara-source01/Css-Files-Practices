6-Inline & Block elements

  • O elemento emlinha leva a largura de seu conteúdo (span, strong, i, em, u,...etc).
  • O elemento bloco leva 100% de largura de seu pai (header, main, section, article, footer, h1...h6, div, p,...etc).
  • ¹Para diferenciar entre eles, observa que bloco leva uma linha própria como uma massa a si mesmo mas emlinha está numa linha, leva a largura de o conteúdo que tem, e precisa de um elemento de separação a ajudar que será em uma linha própria de cada elemento emlinha.
  • ²Para diferenciar entre eles, poderia clicar em qualquer elemento de clica direita e escolher inspecionar, pois bloco leva uma cor de uma lina completa, mas emlinha leva uma cor de cobrir o seu conteúdo somente.
