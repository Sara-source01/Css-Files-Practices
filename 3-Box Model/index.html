3-Box Model

  1. A caixa de conteúdo.
  2. Espaço interior (padding): que separar o conteúdo das bordas.
  3. A caixa das bordas.
  4. Espaço externo (margin): que separar qualquer elemento do outro e das bordas.
Nota: poderia contorlar e ajustar de "Box Model".


Listagem rápida:
  • Border
  • Padding
  • Margin

Border{border-style: solid; border-color: red; border-width: 1px}

  • Bordas de todas as direções.

  • Borda de direção top.

  • Borda de direção left.

  • Borda de direção bottom.

  • Borda de direção right.

  • Borda com medição específica de toda direção.

Tipos de border:(solid, dotted, dashed, doubled,...etc)

  • solid

  • dotted

  • dashed

  • double

Como contorlar de largura e altura de caixa?

O elemento leva um valor padrão de largura de navegador e altura de conteúdo. pois pode usar "width" para controlar a largura de caixa e "height" para controlar a altura de caixa.

  • padding de todas as direções.

  • padding-top


  • padding-bottom


  • padding-left


  • padding-right


  • padding com medição específica de toda direção(top, right, bottom, left).


  • padding de todas as direções com medição específica de (top-bottom) e (right-left).

Margin é um espaço externo não mostra como bordas mas com espaço enter o elemento e o próximo elemento, e colocca espaço das bordas.

  • Margin de todas as direçães.

  • Margin de direção top.

  • Margin de direção bottom.

  • Margin de direção right.

  • Margin de direção left.

  • Margin com medição específica de toda direção.

  • Margin de todas as direções com medição específica de (top-bottom) e (right-left).

  • Todo elemento está na esquerda de navegador(espaço externo). no css3 tem formas várias para fazer o elemento ficou no centro (não o conteúdo), pois poderia usar margin-left = margin-right mas se usa width com valor específico não mostra o elemento no centro. pois coloca "margin-left: auto; & margin-right:auto;" ou"margin: 0 auto" (0=top-bottom, auto=left-right).


  • Nota importante: Margin auto só funciona no left & right de forma horizontal, não funciona de forma vertical (top & bottom).
