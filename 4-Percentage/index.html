4-percentagem, Width, Height


Listagem rápida:
  • Width (largura)
  • Height (altura)
  • Max-Min

1- Width (largura):

  • O valor padrão é um valor de elemento, não obstante se coloca esse valor ou não.
  • O valor padrão de largura é a mesma largura do pai dele como body (navegador).
  • A largura de body quando não contém conteúdo é "1264", poderia conferir de clicar no navegador com clica direita e escolher "inspecionar".
  • Tem elementos não levam a largura de pai dele como "span" (emlinha)

"Span": é um elemento emlinha naõ leva a largura de pai dele.

Testing: h3 leva 50%=500px de largura de pai dele (div).



2- Height (altura):

  • O valor padrão de altura é a altura de conteúdo.
  • Se não tem um conteúdo, a altua vai ser "zero" e não mostrará nada no navegador.
  • A altura de body quando não contém conteúdo é "zero"
O elemento leva a altura dependendo de altura de seu conteúdo.

P vai levar a altura de pai dele (div=200px)

P vai levar 50% de altura de pai dele (div=200px).


Se escrevemos alguma coisa nele ele vai ter altura dependendo de altura de seu conteúdo.

Ou colocamos por exemplo height:50 px; ao invés de 100%;.

3- Max-Min

  • Width: a largura fixa de qualquer elemento, não é efetado de nenhuma alteração seja abriu sua página de tela de notebook tanto tela de celular.
  • Max-width: o máximo largura é possível vai ter, e vai efetar de diminuir de tela ao visualizar, não com o conteúdo.
  • Min-width: o mínimo largura é possível vai ter, e vai efetar de diminuir de tela ao visualizar, não com o conteúdo.
